# 発見した問題

実装中に発見した問題を記録する。

## 優先度の基準
- 高: 継続不可 → status: blocked で中断
- 中: 後で対処が必要、現タスクは継続可
- 低: 軽微、時間があれば対処

## 検出元の種類
- 自動検証: scripts/validate_phase.py が検出（第1層）
- 自動レビュー: scripts/auto_review.py → Codex CLI が検出（第2層）
- 成果物レビュー: /execute 完了時のレビューで発見
- Codex レビュー: /result-review codex で発見
- 実装中: 実装作業中に発見
